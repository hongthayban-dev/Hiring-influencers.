การอัปโหลดวีดีโอขึ้น Google Drive

ต้องสร้าง OAuth Client ID ประเภท "Web application" ใน Google Cloud Console แล้วเพิ่มโดเมนที่ใช้เปิดเว็บนี้ใน Authorized JavaScript origins จึงจะอัปโหลดอัตโนมัติได้ ถ้าไม่ตั้งค่า ระบบจะดาวน์โหลดไฟล์ (เปลี่ยนชื่อตาม ID) ให้อัปโหลดเข้า Drive ด้วยมือแทน

YouTube Data API (ดึงยอดผู้ติดตามอัตโนมัติ)

สร้าง API Key จาก Google Cloud Console แล้วเปิดใช้งาน "YouTube Data API v3"
